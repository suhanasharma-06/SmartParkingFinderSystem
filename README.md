# 🚗 Smart Parking Finder System

A Java-based desktop application developed using Java Swing, JDBC, and MySQL to help users find, book, and manage parking slots efficiently.

## 📌 Features

### 👤 User Panel
- User Registration & Login
- Search Parking Locations
- View Available Slots
- Book Parking Slots
- Cancel Booking
- Give Feedback

### 🛠️ Admin Panel
- Admin Login
- Manage Users
- Manage Bookings
- Manage Slots
- Manage Parking Locations
- View Dashboard Analytics

## 💻 Technologies Used

- Java
- Java Swing (GUI)
- JDBC
- MySQL
- Eclipse IDE

## 🗃️ Database

MySQL database used for:
- User data
- Booking records
- Parking slots
- Feedback
- Admin records
